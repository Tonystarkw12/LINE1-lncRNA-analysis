## 使用指南

### ⚡ 快速开始
```bash
# 1. 环境设置
bash setup.sh

# 2. 激活环境
conda activate line1_lncrna

# 3. 运行分析
jupyter notebook notebooks/LINE1_lncRNA_analysis.ipynb
```

### 📝 数据准备
- 公共RNA-seq数据 (GEO/SRA)
- 参考基因组 (hg38)
- 基因注释 (GENCODE v45)
- LINE-1序列 (RepeatMasker)

### 🔄 自定义配置
修改 `config/config.yaml` 中的参数:
- 差异表达阈值
- 保守性标准
- 网络分析参数
- 输出路径设置
